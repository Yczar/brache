Brache is a modern, intuitive framework for building graphical applications. Whether you're looking to draw shapes, design complex UI components, or animate graphical elements, Brache has got you covered.

## Why Brache?

There are numerous graphics libraries and frameworks out there, so why choose Brache?

- **Simplicity**: Designed with developers in mind, Brache provides a straightforward API to get you started quickly.
- **Flexibility**: From basic shapes to advanced animations, Brache adapts to your needs.
- **Performance**: Optimized for smooth and efficient rendering.

## Core Concepts

Brache is built around a few key concepts:

- **$Component**: The building block for any graphical element in Brache.
- **Scene**: Represents a collection of components.
- **Renderer**: Manages the process of drawing your components onto the screen.

  runApp(MyApp());
